🔤 me・ber辞典

me動詞の基本

me動詞は「〜する」という動作を表す形です。
語根の最初の文字によって形が変わります。

  • baca → membaca：読む
  • tulis → menulis：書く
  • sapu → menyapu：掃く
  • kirim → mengirim：送る
  • masak → memasak：料理する

ber動詞の基本

ber動詞は「状態・活動・自然な動作」を表すことが多いです。

  • kerja → bekerja：働く
  • main → bermain：遊ぶ
  • jalan → berjalan：歩く
  • temu → bertemu：会う
  • lari → berlari：走る
